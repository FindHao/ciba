# ciba 词霸
基于python3 PyQt5写的一个简易划词翻译软件。

测试平台：Debian 9 x86_64

数据源取自金山词霸网页版。

# 功能
## 目前实现的功能

- 划词弹出翻译
- 翻译内容有音标和基本释义
- 弹出后自动发美音

# 安装和使用
`git clone`或者下载项目zip，
```shell
pip3 install -r requirements.txt
```
在项目目录下
```shell
python3 entry.py
```
